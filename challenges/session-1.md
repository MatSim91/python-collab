### 10 Python Challenges Based on Chapter 1

#### 1. Simple Calculator
Ask the user for two numbers and print:

- Their sum
- Their difference
- Their product
- Their quotient

Use int() or float() to convert input.

#### 2. Name Repeater
Ask the user for their name and a number.
Print their name repeated that many times using string replication (`*`).

#### 3. Age in Dog Years
Ask the user for their age and print their age multiplied by 7.

#### 4. Seconds Alive
Ask the user for their age and calculate approximately how many seconds they’ve been alive (age × 365 × 24 × 60 × 60).
Use integer math.

#### 5. String Length Checker
Ask the user to type any sentence.
Print how many characters it contains using `len()`.

#### 6. Greeting Generator
Ask for:
- First name
- Last name

Then print a greeting like:
`Hello, John Smith!`

Use string concatenation.

#### 7. Exponent Practice
Ask the user for a base number and an exponent.
Print the result using `**`.

#### 8. Modulo Challenge
Ask the user for two integers and print the remainder when the first is divided by the second.

#### 9. Favorite Number Doubler
Ask the user for their favorite number.
Convert it to an integer and print double its value.

#### 10. Future Age Calculator
Ask the user for their current age and print how old they will be in:

- 1 year
- 5 years
- 10 years

Use `int()` and string concatenation or `str()`.
